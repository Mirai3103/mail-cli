       - Saves new tokens back to keytar

    6. Create getOutlookUserEmail() function that:
       - Uses Microsoft Graph API /me endpoint to get user email
       - GET https://graph.microsoft.com/v1.0/me with Bearer token
       - Returns the userPrincipalName or mail field
